## 🚀 Overview
This is a web-based application that analyzes user input text and determines sentiment using NLP.

- 💚 Positive
- ❤️ Negative
- 💛 Neutral

Displays a confidence percentage based on sentiment score.

## 🛠 Tech Stack
- React + Vite
- Node.js + Express
- Sentiment (NPM)
- Tailwind CSS
- tsparticles

## ✨ Features
- Real-time sentiment analysis
- Responsive UI
- Animated background
- Confidence score

## ⚙️ Setup
git clone <your-repo-link>
cd sentiment-app
npm install
npm install express sentiment cors

## ▶️ Run
node index.js
npm run dev

Open: http://localhost:5173

## 🧠 How It Works
1. User inputs text
2. Sent to backend
3. NLP processes
4. Result displayed

## 📸 Example
Input: This product is amazing!
Output: Positive (High confidence)
